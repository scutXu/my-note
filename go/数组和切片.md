切片和数组的区别在于[ ]中是否带数字

```
var a [10] int

s0 := a[0:3]
var s1 [] int = a[3:10]
```

切片只是数组的引用，修改一方的数据会影响另一方的值。以下代码相当于创建一个数组并基于数据创建切片

```
s := [] bool {
    true,
    true,
    false
}
```
切片／数组的长度和容量

```
var a [10]int
s := a[0:5]

lenA = len(a)       //10
capA = cap(a)       //10    

lenS = len(s)       //5
capS = cap(s)       //10
```
动态创建切片

```
a := make([]int, 5)     //len(a) = 5, cap(a) = 5
b := make([]int, 0, 5)  //len(b) = 0, cap(b) = 5
```
